Config.json

Here you'll find information on how to configure Visios3D through config.json

What is config.json?

Config.json is a JSON file from which Visios3D reads its configuration.

Where is config.json?

You'll find config.json inside serverRoot/tenantName/wwwroot/configuration, for example C:\inetpub\wwwroot\fakat\wwwroot\configuration

How to configure?

Config.json is a JSON file, so everything you write should adhere to JSON conventions.
Comments are not allowed.
Visit this link to learn more about JSON standard.

General configuration:

Below is a list of properties that can be set in config.json if a property isn't set, the default value will be applied.
